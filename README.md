# norashaalan.net (or .github.io)

Personal academic site for **Nora Shaalan** built with GitHub Pages using the Minimal theme.

- Main content: `index.md`  
- Site settings: `_config.yml`  
- Styles: `assets/css/style.scss`  
- Media (CV, headshot, etc.): `assets/`  
- Additional code and tweaks in `_includes/`

Deploys from the `main` branch via GitHub Pages.
